# tigil-gastos

TigilGastos is a web-based savings tracker that helps students save money for tuition by introducing a locking mechanism. Once a user sets a savings goal and activates the lock, they are restricted from withdrawing their savings until they reach their goal or the target date. This system promotes discipline by preventing impulsive spending while still being simple enough to implement. It provides users with a clear view of their progress through a visual interface such as a progress bar.
